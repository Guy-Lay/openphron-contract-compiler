
## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/idealbridgex/openphron-contract-compiler
   ```

2. Install dependencies:
   
   ```bash
   npm install
   ```
   

3. Set up environment variables:

   -**PORT**: 9002.
   

5. Run the application:

   Start the Server
   ```bash
   npm run dev
   ```
